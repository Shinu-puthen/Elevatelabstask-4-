# Task 4 Dash board design

## Task Overview
This project demonstrates the creation of an interactive sales dashboard for analyzing business performance metrics, customer behavior, and product revenue trends over a multi-year period.

## Objective
To build a comprehensive dashboard that enables data-driven decision-making by visualizing key sales metrics, identifying trends, and highlighting areas requiring strategic attention.

## Process and Methodology

### 1. Data Collection and Preparation
- Gathered raw sales data from various sources.
- Identified key data fields: customer names, product IDs, sales quantities, revenue figures, and timestamps.
- Ensured data consistency and completeness.

### 2. Data Cleaning and Transformation
- Used **Excel Power Query** to clean and transform raw data.
- Handled missing values and standardized data formats.
- Removed duplicates and corrected inconsistencies.
- Created calculated columns for revenue aggregation and percentage calculations.


### 4. Dashboard Design in Power BI
- Imported cleaned data into **Power BI**.
- Designed an intuitive layout with clear visual hierarchy.
- Created interactive slicers for year and month filtering.
- Organized visuals into logical sections: metrics, trends, customer analysis, and product analysis.

### 5. Visualizations Created
- **KPI Cards**: Displayed total revenue and sales quantity prominently.
- **Line Chart**: Showed revenue trends over time to identify patterns.
- **Bar Charts**: Visualized top customers and products by revenue and sales quantity.
- **Data Tables**: Listed detailed breakdowns for deeper analysis.

### 6. Insights and Analysis
- Analyzed customer concentration and identified dependencies.
- Evaluated product performance and categorization quality.
- Identified revenue trend patterns and potential risk areas.
- Formulated strategic recommendations based on findings.

### 7. Documentation and Presentation
- Created a comprehensive PowerPoint presentation summarizing key findings.
- Documented the entire process in this README file.
- Prepared for stakeholder presentations with clear, actionable insights.

## Tools and Technologies Used
- **Microsoft Excel** - Data collection and initial exploration
- **Excel Power Query** - Data cleaning and transformation
- **SQL** - Data aggregation and analysis
- **Power BI** - Dashboard creation and visualization
- **PowerPoint** - Final presentation and reporting
